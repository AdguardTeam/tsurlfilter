## <a id="chromium-mv3-filters"></a> Chromium MV3 filters

These filter lists are used in Chromium MV3 browsers.

### <a id="chromium-mv3-filters-ad-blocking"></a> Ad blocking

#### <a id="chromium-mv3-filters-ad-blocking-adguard-base-filter"></a> AdGuard Base filter

EasyList + AdGuard English filter. This filter is necessary for quality ad blocking.

- Filter ID: **2**
- Path: `dist/filters/chromium-mv3/declarative/ruleset_2/ruleset_2.json`

#### <a id="chromium-mv3-filters-ad-blocking-adguard-mobile-ads-filter"></a> AdGuard Mobile Ads filter

Filter for all known mobile ad networks. Useful for mobile devices.

- Filter ID: **11**
- Path: `dist/filters/chromium-mv3/declarative/ruleset_11/ruleset_11.json`

### <a id="chromium-mv3-filters-privacy"></a> Privacy

#### <a id="chromium-mv3-filters-privacy-adguard-tracking-protection-filter"></a> AdGuard Tracking Protection filter

The most comprehensive list of various online counters and web analytics tools. Use this filter if you do not want your actions on the Internet to be tracked.

- Filter ID: **3**
- Path: `dist/filters/chromium-mv3/declarative/ruleset_3/ruleset_3.json`

#### <a id="chromium-mv3-filters-privacy-adguard-url-tracking-filter"></a> AdGuard URL Tracking filter

Filter that enhances privacy by removing tracking parameters from URLs.

- Filter ID: **17**
- Path: `dist/filters/chromium-mv3/declarative/ruleset_17/ruleset_17.json`

### <a id="chromium-mv3-filters-social-widgets"></a> Social widgets

#### <a id="chromium-mv3-filters-social-widgets-adguard-social-media-filter"></a> AdGuard Social Media filter

Filter for social media widgets such as 'Like' and 'Share' buttons and more.

- Filter ID: **4**
- Path: `dist/filters/chromium-mv3/declarative/ruleset_4/ruleset_4.json`

### <a id="chromium-mv3-filters-annoyances"></a> Annoyances

#### <a id="chromium-mv3-filters-annoyances-adguard-cookie-notices-filter"></a> AdGuard Cookie Notices filter

Blocks cookie notices on web pages.

- Filter ID: **18**
- Path: `dist/filters/chromium-mv3/declarative/ruleset_18/ruleset_18.json`

#### <a id="chromium-mv3-filters-annoyances-adguard-popups-filter"></a> AdGuard Popups filter

Blocks all kinds of pop-ups that are not necessary for websites' operation according to our Filter policy.

- Filter ID: **19**
- Path: `dist/filters/chromium-mv3/declarative/ruleset_19/ruleset_19.json`

#### <a id="chromium-mv3-filters-annoyances-adguard-mobile-app-banners-filter"></a> AdGuard Mobile App Banners filter

Blocks irritating banners that promote mobile apps of websites.

- Filter ID: **20**
- Path: `dist/filters/chromium-mv3/declarative/ruleset_20/ruleset_20.json`

#### <a id="chromium-mv3-filters-annoyances-adguard-other-annoyances-filter"></a> AdGuard Other Annoyances filter

Blocks irritating elements on web pages that do not fall under the popular categories of annoyances.

- Filter ID: **21**
- Path: `dist/filters/chromium-mv3/declarative/ruleset_21/ruleset_21.json`

#### <a id="chromium-mv3-filters-annoyances-adguard-widgets-filter"></a> AdGuard Widgets filter

Blocks annoying third-party widgets: online assistants, live support chats, etc.

- Filter ID: **22**
- Path: `dist/filters/chromium-mv3/declarative/ruleset_22/ruleset_22.json`

### <a id="chromium-mv3-filters-security"></a> Security

#### <a id="chromium-mv3-filters-security-phishing-url-blocklist"></a> Phishing URL Blocklist

- Filter ID: **256**

#### <a id="chromium-mv3-filters-security-ublock-origin-badware-risks"></a> uBlock Origin – Badware risks

Filter for risky sites, warning users of potential threats.

- Filter ID: **257**
- Path: `dist/filters/chromium-mv3/declarative/ruleset_257/ruleset_257.json`

#### <a id="chromium-mv3-filters-security-dandelion-sprouts-anti-malware-list"></a> Dandelion Sprout's Anti-Malware List

- Path: `dist/filters/chromium-mv3/declarative/ruleset_259/ruleset_259.json`
