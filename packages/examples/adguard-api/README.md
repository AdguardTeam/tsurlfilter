# AdGuard API — MV2 Sample Extension

A sample Manifest V2 browser extension demonstrating how to use
[`@adguard/api`](../../adguard-api/README.md) to add content blocking to an
extension. It shows filter list configuration, request blocking, and the
AdGuard Assistant UI.

## Building

```shell
pnpm build
```

The built extension is output to `build/` and can be loaded as an unpacked
extension in Chromium or Firefox.
